## 스터디 방식

- 2주 단위로 주제를 선정(또는 책 범위)한다
- 스터디 시간(온/오프라인)에는 읽은 내용을 바탕으로 각자 파일을 생성하여 내용을 작성한다

  - 이해가 가지 않았던 내용
  - 논의해보고 싶은 내용
- 작성된 내용은 main 브렌치에 푸시해도 되고 PR을 생성해도 된다
- 스터디 시간은 격주 화요일로 고정하나, 일정에 따라 그 주에 다른 요일로 논의 가능하다
